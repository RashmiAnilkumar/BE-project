Dataset

Dataset Collection

The real time dataset from twitter was collected using a tool.The dataset collected has 2970 tweets and each tweet has 19 attributes.Some of the attributes are id_str, from_user, text, created_at, time, user_lang etc.






Dataset Preprocessing

The project focuses on the text attribute of the dataset.The data was preprocessed by removing stopwords, special charaters, numbers,links,tokenization of sentences and removing redundant tweets.The prepocessed data is classified in positive, negative and neutral using the Textblob Sentiment polarity method.
